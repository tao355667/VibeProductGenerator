# VibeProductGenerator - 电商商品主图&文案自动生成工具

一款智能的电商商品内容生成工具，帮助用户快速创建高质量的商品主图和营销文案。

## 🎯 功能特色

### 核心功能
- **商品信息录入**：支持单品录入和批量导入
- **智能主图生成**：自动生成3种不同风格的商品主图
- **文案自动创作**：一键生成简洁版、营销版、详细版三种风格文案
- **任务管理**：完整的任务列表和进度跟踪
- **模板库**：丰富的图文模板供选择
- **个性化设置**：支持自定义生成规则和风格偏好

### 技术特性
- 响应式设计，适配各种屏幕尺寸
- 直观的用户界面，操作简单
- 实时预览生成效果
- 支持图片上传和预览
- 一键复制功能，方便使用
- 模拟AI生成过程，提供真实体验

## 🚀 快速开始

### 环境准备
1. 安装 Node.js（建议 18+）
2. 下载项目后在项目根目录安装依赖：`npm install`

### 密钥配置（仅需配置 ARK_API_KEY）
在项目根目录创建 `.env` 文件，并写入：

```dotenv
ARK_API_KEY=你的火山引擎方舟API密钥
```

> 说明：前端不再保存密钥，所有 AI 请求通过本地 `server.js` 代理转发。

### 启动项目
1. 在项目根目录运行：`npm run start`
2. 浏览器打开：`http://localhost:3000`
3. 开始使用文案生成、图片卖点分析、电商海报生成功能

### 项目结构
```
VibeProductGenerator/
├── .env                            # 环境变量（仅 ARK_API_KEY）
├── server.js                       # 本地代理服务（隐藏 API Key）
├── product-generator-orange.html  # 主应用文件
├── package.json                    # 依赖与启动脚本
├── LICENSE                        # MIT 许可证
└── README.md                      # 项目说明文档
```

## 📖 使用指南

### 1. 商品录入
- 点击左侧导航栏的「商品录入」
- 填写商品基础信息（名称、品牌、类目等）
- 添加商品属性规格（材质、颜色、尺寸等）
- 上传商品参考图片（支持JPG/PNG格式）
- 输入核心卖点（支持回车或逗号分隔）

### 2. 生成内容
- 点击「立即生成草稿」按钮
- 等待2秒模拟生成过程
- 查看生成的主图和文案

### 3. 使用结果
- **主图**：点击图片可放大预览
- **文案**：点击文案卡片可复制到剪贴板
- **重新生成**：不满意可点击「重新生成」按钮
- **清空结果**：可随时清空当前结果

### 4. 高级功能
- **任务管理**：查看和管理所有商品任务
- **模板库**：浏览和收藏图文模板
- **设置**：配置默认生成规则和风格偏好

## 🎨 界面预览

### 主界面
- 侧边栏导航：工作台、商品录入、任务列表、模板库、设置
- 响应式布局：左侧输入区 + 右侧结果区
